# CV Builder - Angular Forms Project

מערכת ליצירת קורות חיים המדגימה שימוש בסוגי טפסים שונים ב-Angular.

## סוגי טפסים שמומשו בפרויקט:

### 1. Template Driven Form
- **מיקום:** טופס פרטים אישיים.
- **נימוק לבחירה:** בחרתי בשיטה זו עבור הפרטים האישיים כיוון שמדובר בטופס סטטי ופשוט יחסית. השימוש ב-`ngModel` מאפשר כתיבה מהירה וקשר ישיר בין ה-Template לנתונים ללא צורך בלוגיקה מורכבת ב-TypeScript.

### 2. Reactive Form
- **מיקום:** טופס פירוט השכלה.
- **נימוק לבחירה:** בחרתי בשיטה זו עבור טופס ההשכלה כיוון שהוא דורש ניהול דינמי של נתונים (הוספה/מחיקה של פריטי השכלה וקורסים). שימוש ב-`FormArray` ו-`FormGroup` מאפשר שליטה מלאה בוולידציות מורכבות וניהול מצב הטופס בצורה תכנותית ומדויקת.

## ולידציות מרכזיות:
- תקינות פורמט אימייל וטלפון.
- תאריך סיום חייב להיות מאוחר מתאריך התחלה.
- ממוצע ציונים בטווח 0-100.
- ולידציות מותנות (תאריך סיום חובה רק אם הסטטוס הוא "סיימתי").
